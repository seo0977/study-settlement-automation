# study-settlement-automation

CPA 스터디그룹의 주간 공부시간 기록 및 보증금 기반 벌금/상금 정산을 자동화하는 프로젝트입니다.

## Background

스터디그룹은 48주 이상 이전 방장이 매주 아래 과정을 엑셀로 수작업 처리해왔습니다.

- 스터디원별 요일당 공부시간 및 상벌점 기록
- 주간 합계 계산 후 기준시간(기본 55시간, 시험 임박 시 임시 상향) 대비 자격 판정
- 자격 미달 + 상벌점 마이너스인 경우 벌금 산정
- 그 주 걷힌 벌금을 자격자끼리 분배, 개인별 보증금 잔액에 반영
- 매주 새 시트를 복사해 위 과정을 반복

이전 방장의 개인 사정으로 제가 정산 및 보증금 관리 역할을 인계받아 직접
정산을 담당하게 되었고, 첫 주 정산을 진행하며 이 반복 작업이 매주 상당한
시간이 소요된다는 것을 직접 체감했습니다.

실제 운영 규정에는 출석·중간인증·반휴·최종 시간 인증 등 다양한 세부 항목이
있었지만, 반복 계산으로 인한 비효율이 집중적으로 발생하는 지점은
"공부시간·상벌점 집계 → 자격 판정 → 벌금/상금 정산" 구간이었기 때문에,
이 부분을 자동화 대상으로 한정했습니다.

## Goal

- 주간 공부시간·상벌점 데이터를 입력하면 벌금/상금 및 잔액 변동을 자동 산출
- 기존 엑셀 정산 로직을 Python으로 재구현

## How it works

`settlement.py`는 실제 운영 규칙을 그대로 코드로 옮겼습니다.

### 1. 보증금 기반 잔액 (`Member`)

전원 보증금 10,000원으로 시작합니다. 이후 매주 정산 결과가 이 잔액에 누적되며,
방장이 별도로 벌금을 걷거나 상금을 지급하지 않고 잔액만 계속 갱신됩니다.

### 2. 주간 처리 (`process_week`)

한 주치 데이터(스터디원별 공부시간, 상벌점)를 입력하면:

- **자격 판정**: 기준 시간 이상 달성 + 상벌점 합계 0 이상 → 그 주 자격자
- **자격 없음 + 상벌점 마이너스**: 벌금 = `|상벌점| x 1,000원`을 잔액에서 즉시 차감
- **자격 없음 + 상벌점 0 이상** (시간만 미달): 벌금 없음, 잔액 변동 없음
- 그 주 걷힌 벌금 총액을 그 주 자격자들끼리 1/n (반올림)하여 잔액에 가산
- 잔액이 마이너스가 되어도 자동 충전 없이 그대로 누적됩니다.

### 3. 중도탈퇴 처리 (`withdraw`)

탈퇴 시 보증금 반납이 없으므로, 탈퇴 시점의 잔액 전액을 잔존 인원에게
1/n(반올림)하여 분배합니다.

### 4. 최종 정산 (`final_settlement`)

각 멤버의 최종 잔액을 계산합니다. 이 금액이 곧 방장이 각자에게 돌려주거나
회수해야 할 금액입니다.

### 실행 예시

\`\`\`bash
python settlement.py
\`\`\`

## Why this matters

방장을 인계받으며 정산 규칙도 함께 재설계했습니다(보증금 기반 잔액 관리, 주간
벌금/상금 즉시 정산, 중도탈퇴 시 처리 기준 명확화). 규칙 자체는 명확했지만
매주 반복 계산하는 과정이 비효율적이라는 걸 직접 운영하며 느꼈고, 동일한 규칙을
코드로 재현해 기준(기준시간·벌금 단가)이 바뀌어도 코드 수정만으로 대응할 수
있도록 구조화했습니다.

## Example Output

\`\`\`
[Week 1] 벌금 총액 500원 / 자격자 6명 / 1인당 상금 83원

- Member1: 61.05h (상벌점 +2) ✅자격 → +83원 (기초 10,000 → 기말 10,083)
- Member2: 50.07h (상벌점 -0.5)  → -500원 (기초 10,000 → 기말 9,500)
- Member5: 64.77h (상벌점 +6) ✅자격 → +83원 (기초 10,000 → 기말 10,083)

===== 최종 정산 (기말잔액 = 방장이 돌려줄 금액) =====

- Member1: 기말잔액 11,271원 (보증금 대비 +1,271원)
- Member3: 기말잔액 11,188원 (보증금 대비 +1,188원)
- Member2 (중도탈퇴): 기말잔액 0원 (보증금 대비 -10,000원)
\`\`\`

기준을 충족한 멤버들은 보증금보다 더 많은 금액을 돌려받고, 중도탈퇴자의 잔액은
전액 몰수되어 잔존 인원에게 재분배되는 것을 확인할 수 있습니다.

`examples/real_data_sample.py`는 실제 정산 시트 데이터를 기반으로 하며,
계산된 벌금/상금 변동액이 실제 기록과 정확히 일치함을 확인했습니다.
